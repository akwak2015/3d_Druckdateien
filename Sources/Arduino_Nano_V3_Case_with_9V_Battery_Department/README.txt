# Summary

Hi there, 
this Arduino Nano V3 case provides plenty of space underneath the actual Arduino. In my case this free space was used to hide a HC05 Bluetooth module.
This case also has an extra department for a 9V Battery, wich can be wired directly to the Vin-Pin of the Arduino. I included the Fusion360 files, if you want to modify the design.
The top and bottom of the case can be combined by screwing a M3 screw in the mounting holes.

####################################
Important: In order to close the case the Arduino ICSP Header has to be desoldered or resoldered the other way around.


Have fun!
