# Summary

Der Sagrotan Verschluss besteht aus dem Stöpsel und dem Grundteil damit er auch verschraubt werden kann. Eine Schablone ist auch dabei (zuerst 16mm Loch bohren) danach Schablone drauf und damit die 5mm Bohrung machen.

Den Deckel hab ich mit Support gedruckt

The Sagrotan closure consists of the plug and the base so that it can also be screwed. A template is also there to have the correct drill distance. First make the 16mm hole, then give the template in the hole and you can make de 5mm hole.
